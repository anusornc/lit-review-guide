# Scholarly search tool research notes

Checked: 2026-07-22

This note records the official sources used to curate the website's scholarly search section. Access models and product features can change, so the website describes them conservatively and links users to the providers.

## Selection principles

- Separate searchable indexes and subject databases from AI-assisted discovery and paper-reading tools.
- Recommend a broad index together with the databases that matter in the researcher's discipline.
- State when full text, export, saved searches, or advanced features depend on an account or institutional subscription.
- Do not imply that citation counts, AI rankings, or inclusion in an index establish study quality.
- Warn when a source contains preprints or primarily searches deposited metadata rather than full text.

## Important limitations reflected in the website

- Google Scholar's public help states that only up to 1,000 results are visible for a query and that bulk access is unavailable; it should not be treated as a complete protocol search by itself.
- OpenAlex and other large aggregators can contain incomplete or incorrect metadata and automated labels; verify identifiers and publication status at the source.
- PubMed indexes citations and abstracts rather than guaranteeing full text, and no single subject database covers every relevant source.
- arXiv records may not have passed peer review and can change between versions.
- Scopus, Web of Science, IEEE Xplore, ACM Digital Library, JSTOR, Project MUSE, and some Dimensions or Lens features depend on the researcher's account or institution.
